## 🕓 앱 소개

주파수를 이용하여 모기 퇴치를 돕는 앱입니다.
또한 내가 원하는 볼륨과 주파수로 조절 가능합니다.

## 🖍️ 주요 기능

- 주파수 조절
- 볼륨 조절
